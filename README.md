## Installation


Clone the repo:

```bash
git clone --depth 1 https://github.com/hagopj13/node-express-boilerplate.git
cd node-express-boilerplate

```

Install the dependencies:

```bash
pip install plotly
pip install numpy
pip install pandas
pip install scipy
pip install yfinance
```
